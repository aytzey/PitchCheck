# Code of Conduct

PitchCheck follows the Contributor Covenant Code of Conduct.

## Expected Behavior

- Be direct, constructive, and respectful.
- Assume technical disagreement is about the work, not the person.
- Keep issues and pull requests focused on reproducible facts.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing another person's private information or credentials.
- Deliberately disrupting maintainers, users, or contributors.

## Enforcement

Maintainers may moderate, edit, or remove comments and issues that violate this
code of conduct. Serious or repeated violations may result in a ban from the
project community.
